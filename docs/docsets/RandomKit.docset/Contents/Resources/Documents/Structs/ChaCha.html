ChaCha

public struct ChaCha: RandomBytesGenerator, Seedable, SeedableFromSequence, SeedableFromRandomGenerator

A generator that uses a ChaCha20 algorithm. There is also a Wikipedia section describing this algorithm.

This implementation is taken from that of ChaCha in the Rust rand crate.
